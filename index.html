Seinfeld Quiz

The brains behind this quiz were coded with Javascript and jQuery. All of the questions and and options are stored in a 3-D array where they are dynamically added to the DOM when the user clicks the 'Next' button. The user also has the option to cycle back and forth between questions just in case they change their mind. When the quiz ends, the users score is calculated and shown and there is an option to retake the quiz.
